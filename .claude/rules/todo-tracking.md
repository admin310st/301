# TODO Tracking (301.st)

## wiki/TODO.md — единый чеклист

* Каждый план из `~/.claude/plans/` (касающийся 301) должен быть отражён в `wiki/TODO.md`.
* Формат: секция с заголовком задачи + чеклист `- [ ]` / `- [x]`.
* После полной реализации задачи — удалить секцию из TODO (не оставлять выполненные).
* TODO содержит только открытые вопросы и незавершённые задачи.
* Не дублировать контент плана целиком — только краткий чеклист и ссылку на план.
